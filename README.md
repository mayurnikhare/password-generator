# 🔐 Password Generator (Python)

A secure and customizable password generator built using Python. This tool allows users to generate strong passwords based on length and character preferences.

---

## 🚀 Features
- Generate passwords of any length
- Option to include:
  - Uppercase letters (A-Z)
  - Numbers (0-9)
  - Special symbols (!@#$...)
- Uses Python's `random` and `string` libraries
- Simple and user-friendly CLI interface

---

## 🛠️ Technologies Used
- Python
- random module
- string module
